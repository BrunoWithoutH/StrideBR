## Access
**Visit the project page at:** [https://stridebr.alwaysdata.net](https://stridebr.alwaysdata.net)


## Features
Current and upcoming features include:

- [x] Physical activity tracking  
- [x] Weekly training schedule  
- [x] Workout tools  
- [ ] Sports event calendar
